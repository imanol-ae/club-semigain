Alta de nuevo usuario/a de la web

Para poder darte de alta en la web, primero debes contactar con el encargado del club para que un administrador te registre como jugador del club. El mismo encargado te proporcionará el número de socio con el que podrás darte de alta en la web y realizar las reservas de instalaciones.

Correo electrónico Campo obligatorio Introduce una dirección de correo válida. La dirección de correo es necesaria. Número de socio Campo obligatorio El número de socio es como mínimo de 4 caracteres. El número de socio es como máximo de 4 caracteres. El número de socio debe contener 3 números y una letra mayúscula. El número de socio es necesario. Contraseña Campo obligatorio La contraseña debe contener como mínimo 6 caracteres. La contraseña debe contener entre 8 y 16 caractéres, y al menos, una letra en minúscula, otra letra en mayúscula, un número y un caracter especial. Las contraseñas no coinciden. Repetir contraseña Campo obligatorio Las contraseñas no coinciden.
